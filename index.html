Мой первый сайт

Этот сайт я сделал самостоятельно в рамках задания по информатике. Цель была — понять, как создаются сайты и как HTML работает на практике.

Почему я решил сделать сайт

Мне стало интересно, как обычные страницы в интернете открываются в браузере и из чего они состоят.

Как был создан этот сайт

  • Я использовал язык разметки HTML
  • Создал файл index.html
  • Добавил заголовки, текст и список
  • Разместил сайт в интернете с помощью GitHub Pages

Сайт создан мной самостоятельно.
